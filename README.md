# code-sync-tool
##### 
用于在项目中拉取自有model的命令行工具

English | [简体中文](./README.zh_CN.md) 

## Install

```bash
$ npm i code-sync-tool -s -g
```

## Usage

```node
// 在项目根目录执行以下命令 moduleName为要安装的包名
  code-sync add `moduleName`
```
